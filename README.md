### Description

This bot serves to help me learn Go.

### Initialization

1. Create a file in the same directory of `main.go` called `init.txt`
2. The `init.txt` file should look exactly as follows (ignore any square brackets):

    ```
    TOKEN: [Bot token]

    AUTORESPONSES
    [Prompt 1]/[Response 1]
    [Prompt 2]/[Response 2]
    ```
